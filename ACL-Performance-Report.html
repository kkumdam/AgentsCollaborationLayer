7. Core Findings

1. ACL achieves O(n) total token cost vs NL O(n²) — the efficiency gap widens exponentially at scale.
2. Typed state transitions eliminate format drift, reducing error cascades that plague NL handoffs at depth >5.
3. Content-addressed artifacts (artifact://type/sha256) avoid re-serializing data between agents, cutting redundant tokens.
4. Policy-governed spawning prevents runaway costs — budget caps, TTL, and tool-access inheritance are enforced at the protocol level.
5. At production scale (10K pipelines/day), ACL saves an estimated $207,242/year in token costs alone, with +29% reliability gain.
6. High-cost models (Claude Opus, GPT-4o) benefit most — ACL saves $175.95 per 1K handoffs for Claude Opus.
